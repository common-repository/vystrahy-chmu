== Description ==

! 29. KVĚTNA 2019 ČHMÚ UKONČIL VYDÁVÁNÍ VÝSTRAH SIVS A PŘEŠEL VÝHRADNĚ NA FORMÁT CAP. PROTO SE MOMENTÁLNĚ NEZOBRAZUJÍ ŽÁDNÉ VÝSTRAHY. NA PŘEPRACOVÁNÍ PLUGINU PRO FORMÁT CAP SE PRACUJE.

Jednoduchý plugin, který přidává widget co zobrazuje výstrahy vydané Českým hydrometeorologickým ústavem.

A simple plugin that adds widget with warnings of Czech Hydrometeorological Institute.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/vystrahy-chmu' directory, or install the plugin through the WordPress plugins screen directly
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Add the 'Výstrahy CHMÚ' to your widget area
1. Set the widget
